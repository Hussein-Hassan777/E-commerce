# E-commerce Web Application

A full-stack E-commerce web application built with **HTML, CSS, JavaScript, and PHP**. This project allows users to browse products, add items to their cart, make payments, and for admins to manage the store.

## Features

### User Features

- **Main Page** – Displays available products with images, descriptions, and prices.
- **Product Details / More Info** – View detailed information about each product.
- **Shopping Cart** – Add, remove, and update products before checkout.
- **Payment Integration** – Complete purchases securely through a payment interface.

### Admin Features

- **Admin Dashboard** – Add, edit, and delete products.
- **Order Management** – View user orders and track status.

## Technology Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL

## Installation / Setup

1. Clone the repository:

```
git clone https://github.com/Hussein-Hassan777/E-commerce.git
```

1. Place the project folder in your local server directory (e.g., `htdocs` for XAMPP).
2. Create a database and import the SQL file (`ecommerce_db.sql`) included in the project.
3. Configure database connection in `config.php`.
4. Start your local server and open the project in your browser.

## Usage

- Navigate the main page to view products.
- Click **More Info** on a product to see details.
- Add items to **My Cart** and proceed to **Payment**.
- Admin can log in to the dashboard to manage products and view orders.
